h1. ActsAsLink

h2. Installation

pre. gem install acts_as_link

h2. Example

<pre>
require 'rubygems'
require 'acts_as_link'

link = Link.new('http://google.com')
link.is_broken?     #false
link.is_an_image?   #false
</pre>
